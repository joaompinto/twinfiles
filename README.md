## Motivation

Over the years I have accumulated multiple backups of photos leading to a lot of duplicates. I need an easy way to clean my photos.

## Installation

```bash
pip install twinfiles
```

## Usage

```sh
# To delete add --delete
twinfiles ~/OneDrive/Pictures
```
